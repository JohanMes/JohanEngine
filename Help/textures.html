class Textures

Description

A singleton that creates and destroys textures.

Functions

  • Constructor()

    Called once by InitEngine.

  • Destructor()

    Called once by DeleteEngine.

  • Texture* Add(const char* path)

    Add texture from file. Returns first texture on pile or NULL if pile is empty on failure. When adding duplicate, returns pointer to first addition.

  • Texture* GetByFullPath(const char* fullpath)

    Returns texture with equal full path, or NULL if not found

  • Texture* GetByFileName(const char* filename)

    Returns texture with equal file name, or NULL if not found

  • void Delete(Texture* thistexture)

    Deletes texture by pointer. Ignores NULL input.

  • void Clear()

    Clears whole database and frees all valid pointers.

Members

  • std::list<Texture*> list

    Linked list of all textures.
