class Models

Description

A singleton that creates and destroys models.

Functions

  • Constructor()

    Called once by InitEngine.

  • Destructor()

    Called once by DeleteEngine.

  • Model* AddFromBuffer(VERTEX* vb,unsigned int* ib,unsigned int numvertices,unsigned int numindices,unsigned int numfaces)

    Creates model from provides arrays by copying them. Also generates bounding sphere and sends buffer to GPU.

  • Model* AddLocalFromOBJ(const char* path,bool optimize);

    Creates local model from Wavefront OBJ file. Removes duplicate VERTEX entries when optimize is true. Local models are not sent to the GPU by Model::CreateBuffers.

  • void Delete(Texture* thistexture)

    Deletes textures by pointer. Ignores NULL input.

  • void Clear()

    Clears whole database and frees all valid pointers.

Members

  • std::list<Texture*> list

    Linked list of all textures.
