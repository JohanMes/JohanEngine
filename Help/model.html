class Texture

Description

Stores a single texture in LPDIRECT3DTEXTURE9 format. Create one by calling Textures::Add().

Functions

  • Constructor(const char* texpath)

    Not to be used. Please call Textures::Add().

  • Destructor()

    Not to be used. Please call Textures::Delete().

  • bool GetInfo(D3DSURFACE_DESC* result)

    Returns information about texture in D3DSURFACE_DESC struct. Returns false on failure.

  • int GetWidth()

    Returns width of texture in pixels, or -1 on failure.

  • int GetHeight()

    Returns height of texture in pixels, or -1 on failure.

Members

  • std::list<Texture*>::iterator bufferlocation

    Contains location of texture in list of Texture singleton.

  • LPDIRECT3DTEXTURE9 pointer

    Pointer to Direct3D texture object.

  • char* fullpath

    Pointer to full path to texture file.
